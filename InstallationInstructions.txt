Installation Instructions

Installing the lbrary and command-line runner:

 Run the install_storevil.bat batch file, passing the directory to which to install, for example:

 install_storevil.bat C:\StorEvil

 To uninstall StorEvil, just delete the folder you installed it to.

Installing the ReSharper runner:

 run the install_resharper.bat batch file
 
Uninstall the ReSharper runner:

 run the uninstall_resharper.bat batch file
